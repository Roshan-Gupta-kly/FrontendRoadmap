Frontend Development

CSS JavaScript

Frontend development is the development of the graphical user interface of a website, through the use of HTML, CSS, and JavaScript so that users can view and interact with website.

  • HTML: HTML stands for HyperText Markup Language. It's the standard markup language for creating web pages and web applications. HTML provides the structure and content of a webpage by using a system of tags and attributes to define various elements such as headings, paragraphs, links, images, forms, and more.


  • CSS: CSS stands for Cascading Style Sheets. It's a style sheet language used to describe the presentation of a document written in a markup language like HTML.

    1. Pure CSS

    2. Bootstrap (Framework)

    3. Tailwind (Framework)

    4. Sass (Framework)



  • JavaScript: JavaScript JavaScript is a high-level, interpreted programming language primarily used to add interactivity and dynamic behavior to web pages.

    1. Vanilla JS

    2. jQuery (Framework)

    3. React (Framework)

    4. Angular (Framework)

    5. Vue (Framework)

    6. Typescript (Preprocessor)
